# LMC e-learning システム

LMC（合同会社ライフマネジメントカンパニー）の経営・営業講座向け自社e-learningシステム。
将来的にパッケージ製品として外販可能な構成。

## 機能（詳細設計書 v1.0 準拠）

- 会員: ログイン（有効期限チェック）／コース目次（契約プラン別・検索・お気に入り）／動画視聴（YouTube限定公開・視聴回数記録・最後まで再生で視聴完了）／確認テスト（ランダム5問・4問で合格・再挑戦無制限）／達成率ダッシュボード
- 管理者: レッスン管理（追加・削除・並べ替え・公開切替）／問題管理／プラン・コース管理／会員管理（発行・有効期限・仮PW再発行）／進捗管理
- カラーテーマ2種（スタンダード：ネイビー×ゴールド／ローズ）・PC/スマホ対応

## 技術構成

Next.js (TypeScript) + Supabase (PostgreSQL・東京リージョン) + Vercel + YouTube限定公開

## 開発

```bash
npm install
npm run dev   # http://localhost:3000
```

現在は**プロトタイプ（デモモード）**: データはブラウザのlocalStorageに保存。
ログイン画面のデモアカウントボタンから会員／管理者を切替できる。

## Supabase接続（本番化手順）

1. Supabaseプロジェクト作成（**東京リージョン**）
2. SQL Editor で `supabase/schema.sql` を実行
3. `.env.local.example` を `.env.local` にコピーしてURL・anonキーを設定
4. `lib/store.tsx` をSupabase実装に差し替え（データ操作はこの1ファイルに集約済み）
5. RLS検証（未ログイン0件／会員は自分のみ＋プラン内コースのみ／管理者は全件）
